# Soul Panther - Official Landing Page

Sacred futurism meets independent electronic music.

## About

Soul Panther is an independent electronic music artist exploring the frontier where music, consciousness, and AI meet. This is the official landing page showcasing music, videos, and booking information.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Fonts:** Archivo Black (display), Work Sans (body)
- **Deployment:** Vercel

## Features

- Responsive design optimized for all devices
- YouTube video embeds
- Spotify integration
- Booking contact form
- Newsletter signup
- Premium sacred futurism aesthetic

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
